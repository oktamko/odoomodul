E Commerce Hero Area Menu

Add a Hero Menu with Slider to Your Odoo Website

How to Install and Use the E Commerce Hero Area Menu Module

Follow these simple steps to add a hero menu to your website:

  1. Log in to your Odoo instance as an administrator.
  2. Click on the "Apps" menu item and search for "E Commerce Hero Area Menu".
  3. Click on the module and install it.
  4. Go to the page where you want to add the hero menu and click the edit button.
  5. Click on the "Snippets" option or search for "Hero Area Menu".
  6. Select the hero menu snippet to add it to your website.

The E Commerce Hero Area Menu module is fully customizable, allowing you to effortlessly adjust it to fit your website's unique design and branding. With this module, you can customize the colors, fonts, and other visual elements using CSS.

If you have any questions or issues with the module, please refer to the documentation or contact the developer.

Output Example

The hero menu provides an eye-catching way to highlight your website's main content and grab the user's attention. With this module, you can effortlessly create a hero menu that includes text, images, and calls to action.
